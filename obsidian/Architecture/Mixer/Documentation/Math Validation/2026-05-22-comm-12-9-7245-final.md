# Final Verdict — B(2,5) comm_12_9 7,245-char reduction

## The claim

The 7,245-char word at `experiments/b25_reduce_core/corrected/final_beam.txt` represents the same element of B(2,5) as `[c12, c9]`, where `c_k = [[a,b], a^{k-1} b a^{-(k-1)}]`.

## Sub-claim A — Input word is a B(2,5) identity

**Status: VERIFIED**

The 28,652-char benchmark word at `experiments/b25_reduce_core/benchmark_run/words/comm_12_9.txt` matches the `comm_12_9` entry in `b25_relators_havas_verified.txt` byte-for-byte. That file contains the 153 relators from the Hall–Sims–Havas computer proof of B(2,5)'s finiteness — verified B(2,5) identities. The 28,652-char input word IS the identity in B(2,5).

## Sub-claim B — Reduction chain preserves group equality

**Status: PARTIALLY VERIFIED**

- **Abelianization preserved (0,0) at all 8 stages**: confirmed independently by GAP 4.15.1 for the endpoints (w1 and w2), and documented by Experimenter-B25 in `experiments/b25_reduce_core/corrected/README.md` for all intermediate stages. This rules out the class of bugs that invalidated the previous results (word_3707/3732/4882 had abelianization (4,4)).
- **Lifted-KB rule m → m_pure**: rule exists in `runs/b25_lifted/20260330_132357/kb_rpo/input.kbprog.live`. KB-derived rules are sound by construction (V3 in [[verification-methods-for-group-equality]]). This supports the coreless expansion step.
- **Full step-by-step rule-application verification**: not completed. Each step type (KB rules, power reduction, braid reduction) is individually sound by construction, but independent re-execution of the full chain from `experiments/b25_reduce_core/corrected/` was not performed.

## Sub-claim C — Element equality 28,652-char =_G 7,245-char in B(2,5)

**Status: NOT VERIFIED — tooling limit reached**

GAP 4.15.1 with kbmag was run. Checks attempted:

| Check | Result |
|-------|--------|
| Abelianization (GAP) | PASS — (0,0) for both words |
| kbprog rule-bank reduction (GAP Read()) | BLOCKED — 765k-line file exhausts GAP's parser |
| Full B(2,5) `IsOne` | Not attempted — known infeasible; |G| = 5^34 |
| Lifted quotient `IsOne` for m = m_pure | DIVERGED — partial relators make quotient infinite |

KB on B(2,5) does not complete. This is a settled fact in the group theory literature. No script, time-limited run, or quotient trick resolves element equality in B(2,5) without completing KB — which would solve the open problem our experiment program exists to address. The tooling limit on Sub-claim C is a ceiling on the entire B(2,5) word equality problem, not specific to this claim.

## Verdict

**#status/conjectured** — stronger tooling exhausted

The available evidence is consistent with the claim and rules out the abelianization bug class. The input word is verified as a B(2,5) identity (Sub-claim A). The chain's abelianization invariance is confirmed, and the critical m = m_pure step is supported by a KB-derived rule. But element equality in B(2,5) — Sub-claim C — is not reachable by current tools.
