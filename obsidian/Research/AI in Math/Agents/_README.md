## Research/AI in Math/Agents/

**What belongs here:** Paper summaries for work where the primary methodological contribution is an **agentic or LLM-driven system that does mathematics** — specifically:

- LLM-driven formal theorem provers (sampling over proof tactics in Lean, Isabelle, Coq, etc.)
- Agentic proof search (MCTS, tree search, iterative refinement with LLM policies)
- Autoformalization scaffolding (informal NL proof → formal verification pipeline)

**Does not belong here:** RL-for-math work (→ `Research/AI in Math/RL/`), ML-guided algorithm selection or conjecture generation (→ `Research/AI in Math/ML/`), pure math LLM benchmarks without a new agent architecture.

**Filename convention:** arXiv ID where available (`2009.03393.md`); for tech reports without arXiv, use `<first-author>-<year>-<slug>.md`.
