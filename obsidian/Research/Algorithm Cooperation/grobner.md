# Quotient Tests and Gröbner Bases

Kreuzer, Myasnikov & Rosenberger (2025 draft). Proposes Gröbner bases for studying the word problem and infinitude of finitely presented groups via universal linear representations. Directly relevant to the Mixer: the Gröbner-basis word-problem algorithm is structurally a different solver for the same search space that KB explores.

## Abstract

> No verbatim abstract available — source is local PDF at `docs/papers/grobner.pdf`. See TL;DR below.

## TL;DR

For any finitely presented group G = ⟨X | R⟩, constructs a "universal linear representation" μ: G → SL(n, Q_R) where Q_R is a polynomial ring quotient built from the relators. A Gröbner basis of the relator ideal I_R lets one (1) test the word problem: if μ(w) ≢ I after reduction, then w ≠ 1 (semi-decidable, "No"-only); (2) test infinitude: if a matrix image has zero minimal polynomial, G is infinite. Applied to Kourovka problems 7.25 and 11.10b; B(2,5) is proposed but incomplete (Section 7.5: "TODO: Continue!").

## Problem

Given a finitely presented group G = ⟨X | R⟩, can Gröbner basis computation in a polynomial ring derived from the relators:
1. **Certify non-identity** of specific words (word problem, "No" direction)?
2. **Certify infinitude** of the group?

These are two of the hardest computational problems for FPGs; both are undecidable in general. The paper proposes a tractable partial algorithm for specific instances.

## Approach

**Step 1 — Universal linear representation:**

For G = ⟨x₁,...,xₛ | R⟩, introduce indeterminates T = {y^(k)_{ij} | k=1..s, i,j=1..n}. Define the ideal I_R ⊂ ℤ[T] generated by:
- det(x^μ_k) − 1 = 0 (det-one condition for each generator)
- (r^μ)_{ij} − δ_{ij} = 0 for each relator r ∈ R

The universal representation ring: Q_R = ℤ[T]/I_R. Map μ: G → SL(n, Q_R) by x_k ↦ (y^(k)_{ij}).

**Step 2 — Algorithm 5.1 (Universal Linear Quotient Test):**
1. Compute S_R (polynomial equations from relators).
2. Compute Gröbner basis of I_R in ℚ[T].
3. Compute μ(w) symbolically (matrix product of generators).
4. Reduce each entry f_{ij} − δ_{ij} modulo the Gröbner basis.
5. Non-zero remainder → **"No"** (w ≠ 1 in G). Zero remainder → **"Don't know."**

**Step 3 — Algorithm 6.2 (Minimal Polynomial / Infinitude Test):**

For W ∈ Mat(n, K) where K = F[y₁,...,yₘ]/I (affine algebra), form the module U from W−z·Iₙ and I·Kⁿ, compute the annihilator in F[y₁,...,yₘ,z] by Gröbner methods. If minimal polynomial is 0 → W has infinite order → G is infinite.

**Optimization (Remark 3.1):** Split each relator r = r′ ∘ r″ and encode as r′^μ = (r″^{-1})^μ. Halves the polynomial degree (from deg(|r|) to deg(|r|/2)).

## Key result

**Worked results (Kourovka problems):**

- **Problem 7.25 groups G_i (i=5,7,9,11):** Using n=2 with truncated Gröbner basis, confirmed all four are commutative → finite of order 2i. (Verbatim: "G_5, G_7, G_9, G_11 are all commutative, hence finite.")
- **Problem 11.10b groups H_i (i=5,10,15,20):** n=2 suffices to certify a ≠ 1 in H_5, H_10, H_15, H_20.

**B(2,5) (Section 7.5): INCOMPLETE.** The paper proposes studying B(2,5) via μ: B(2,5) → SL(2, Q_R). The section is labeled "TODO: Continue!" — no Gröbner basis results for B(2,5) are reported.

**Key formula:**

Universal linear representation (Proposition 4.2): Q_R = ℤ[T]/I_R; μ(x_k) = (y^(k)_{ij}). Every SL(n,K)-representation of G factors through μ via a ring map Q_R → K.

## Assumptions

- Gröbner basis computation for I_R terminates (not guaranteed — doubly exponential worst case; tractable only for small n and short relators).
- The polynomial encoding correctly captures the group structure (follows from the universal property of Q_R).
- For B(2,5): the paper uses the presentation ⟨a,b | w^5=1 for |w|≤4⟩ (finitely many relators), not the infinite presentation ⟨a,b | w^5=1 for all words w⟩. These differ — see Limitations.

## Limitations / scope

- The word problem test is **semi-decidable**: certifies "No" (w ≠ 1) but never "Yes" (w = 1). For B(2,5) where finiteness is unknown, a "Don't know" is uninformative.
- **Section 7.5 is incomplete:** The paper does not report Gröbner basis results for B(2,5). The framework is proposed, not executed.
- Gröbner basis complexity: doubly exponential worst case. For B(2,5) relators of length ≤ 20, the system S_R at n=2 may be computationally intractable.
- The paper uses the 2002 Kourovka Notebook edition [ref 4]; problem numbers may differ from [[kourovka-2022]] (2022 edition).
- **Not a peer-reviewed paper** — draft as of October 2025; marked "TODO" sections signal incompleteness.

## Why this paper matters

**The Mixer connection is direct and explicit.** Alexei Myasnikov co-authors this paper AND leads the Mixer build ([[problems-people]]). This paper is theoretical backing for a new Mixer agent type: a Gröbner-basis solver cooperating with KB agents on the same FPG presentation.

**Three Mixer integration points** (to be populated as Concepts/ hubs in F4.4):

1. **Gröbner quotient filter** ([[Concepts/grobner-quotient-filter]]): Use the SL(2, Q_R) quotient test to prune dead-end words early — a fast "No" oracle before heavier KB exploration. Structurally identical to the B(4,3) fast-fail role of the second ordering agent.

2. **Gröbner infinitude probe** ([[Concepts/grobner-infinitude-probe]]): Run the minimal-polynomial test on suspected-infinite groups before committing to full KB completion. If the test confirms infinitude, KB can be abandoned early.

3. **Representation variety** ([[Concepts/representation-variety]]): The variety V_R of SL(n,K) representations as a coordinate system on the group's structure — potentially useful for ordering/scoring KB rule injection.

The KB–Gröbner duality is the core Mixer insight: both KB completion and Gröbner basis computation produce canonical rewriting systems — one in string rewriting, one in polynomial arithmetic. A Mixer combining both could share structure across the two domains.
