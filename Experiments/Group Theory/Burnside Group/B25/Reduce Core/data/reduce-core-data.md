## The X core (key constant)

```
X = ABabAbabABaBAbabABABabAbaBABaBAbabb  (35 chars)
```

X⁻¹ = (reverse and invert each char of X).

m substitution: `m = X⁻¹bX` (71 chars when expanded). Used for lifting comm_12_9 (28,652 → 544 chars). Verified: X appears 34 times in comm_13_10 (length 2500).
